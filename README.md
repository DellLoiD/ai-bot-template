# Telegram Bot для управления регистрацией пользователей

Этот Telegram-бот предназначен для управления регистрацией пользователей с использованием кодов доступа. Он позволяет администратору генерировать коды, отслеживать зарегистрированных пользователей, проводить рассылки и экспортировать данные.

## Основные функции

- **Регистрация пользователей** по коду доступа
- **Админ-панель** для управления ботом
- **Генерация кодов доступа**
- **Просмотр всех пользователей**
- **Экспорт данных в Excel**
- **Массовая рассылка сообщений**
- **Управление пользователями** (удаление)

## Команды бота

### Общие команды (для всех пользователей)

- `/start` — начать взаимодействие с ботом
- `🔑 Ввести код` — ввести код доступа для регистрации
- `📅 Проверить регистрацию` — проверить статус регистрации

### Админ-команды (только для администратора)

- `/admin` — открыть админ-панель
- `📋 Все пользователи` — посмотреть список всех зарегистрированных пользователей
- `📊 Выгрузить Excel` — экспортировать базу данных пользователей в Excel-файл
- `📤 Рассылка` — начать процесс массовой рассылки сообщений
- `🔑 Сгенерировать коды доступа` — сгенерировать 10 новых кодов доступа
- `🔑 Коды доступа` — посмотреть список доступных (неиспользованных) кодов
- `🗑️ Удалить пользователя` — удалить пользователя из базы данных по его ID

## Использование

1. Запустите бота с помощью команды:
   ```bash
   python main.py
   ```

2. Администратор может войти в админ-панель с помощью команды `/admin` и использовать соответствующие кнопки для управления.

3. Обычные пользователи должны ввести код доступа, чтобы пройти регистрацию и указать свои данные (имя, дату рождения, пожелания).

## Структура проекта

- `main.py` — точка входа, запуск бота
- `config.py` — конфигурация (токен, ID администратора, настройки)
- `loader.py` — инициализация компонентов бота
- `database/` — работа с базой данных
  - `data.py` — класс Database для работы с SQLite
  - `methods.py` — методы для работы с кодами активации
- `handlers/` — обработчики команд и сообщений
- `keyboards/` — клавиатуры (обычные и админские)
- `states.py` — состояния для FSM
- `utils/` — вспомогательные утилиты
- `users.db` — база данных пользователей
- `.env` — переменные окружения

## Настройка

1. Создайте файл `.env` в корне проекта:
   ```env
   BOT_TOKEN=укажитетокен вашего бота
   ADMIN_ID=укажите id вашего телеграмма
   DB_NAME=users.db
   WELCOME_TEXT=Добро пожаловать в нашу систему!
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите бота:
   ```bash
   python main.py
   ```
